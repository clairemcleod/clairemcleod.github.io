Mauck Lab Data Tools



Image Blinding (Python)

Python script that copies images and assigns a random number filename to the copy (for blinded image analysis). Allows user to add more images at a later time without identifier duplication

Repository (public)


Nuclear Aspect Ratio Quantification (Matlab)

Processes fluorescent images as cells are progressively strained to measure change in aspect ratio. Also calculates lagrangian strain between cell centroids.

Repository (private)


Atomic Force Microscope Curve Fitting (Matlab)

Fits Hertz model to AFM curves, for both spherical and pyramidal tips. Rudimentary contact point determination via goodness-of-fit optimization.

Repository (private)


Image Analysis: Radial Profiles (ImageJ Macro + R)

ImageJ macro automatically measures and records intensity profiles emanating from a user-identified location (e.g. cell center). Batches processes all images in directory, and ouputs profile intensities in long format in csvs. Example R analysis included.

Repository (public)
